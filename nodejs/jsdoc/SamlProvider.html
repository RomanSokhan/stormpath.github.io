Class: SamlProvider

SamlProvider

new SamlProvider(samlProviderResource)

Encapsulates a SamlProvider resource, which is a type of Provider. For full documentation of the Provider resource, please see REST API Reference: Provider.

For a high-level overview of SAML Authentication, please see Authenticating Against a SAML Directory.

For more information about configuring SAML Authentication, please see Configuring SAML via REST.

This class should not be manually constructed. It should be obtained from one of these methods:

  • Directory.getProvider()
