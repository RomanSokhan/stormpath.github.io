StormpathError

public class StormpathError: NSError

StormpathError represents an error that can be passed back by a Stormpath API after a network response. It typically contains a HTTP status code, along with a localizedDescription of the specific error as passed back by the Framework Integration API. It also has two special error codes:

0 - Internal SDK Error (which should be reported to Stormpath as a bug) 1 - Unrecognized API Response (which means the Framework integration may not support this version of Stormpath)
